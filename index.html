Red-Black-Tree Data Structure Project:

A tree data structure that rebalances "automatically" and allows O(log(n)) add, find, and remove operations that is faster than the AVL tree by using recurrsion and rebalancing the tree during the downward traversal.
RB Tree repository

Hash Table Data Structure Project:

A data structure using a hash function to store data in a hash table that uses separate chaining to handle collisions and resizing the table to keep add, find, and remove operations to O(1).
